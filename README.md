## 介绍


本项目是基于[vue-admin-template](https://gitee.com/panjiachen/vue-admin-template)
模板进行改造的一个博客后台管理项目,目前是用于自己的博客管理。
有不懂的问题可以随时私聊或者前往作者博客地址首页QQ图标与我联系。
由于七牛云图片没有设置https，gitee无法添加图片介绍，可以前往博客首页查看文章介绍。另外欢迎star

## 新增功能
- 文章贡献度，类似github的提交表格
- 操作日志，记录用户在门户页面的操作记录，

- 文章SEO

- 友链管理
